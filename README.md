# study.track

A minimal project-based study tracker. Add projects, list what you need to learn for each one, and check topics off as you go.

## Features

- Create projects with name + description
- Add study topics per project (with explanation of why you need them)
- Check off topics as you complete them
- Progress bar and stats per project
- Everything saved to localStorage — no backend needed

## Usage

Just open `index.html` in a browser. No build step, no dependencies, no server.

Or host it anywhere static (GitHub Pages, Netlify, etc.)
